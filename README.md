# goatpy
Spatial Glycomics Analysis Toolkit

## Install

To install goatpy please follow the instructions for creating a conda enviornemnt. You will need to download the `environment.yml` file and store it where you want ur environemnt to be kept.

```
conda env create -f environment.yml
```

Once created we can activate the environment:

```
conda activate myenv
```
And install goatpy

```
pip install git+https://github.com/agc888/goatpy.git
```
